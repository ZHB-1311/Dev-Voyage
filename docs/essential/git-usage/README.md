# 版本控制基础

::: tip 写在前面
如果你曾在深夜因为改崩代码而手足无措，或者经历过"最终版"、"最终版2"、"最终版真的不改了"的文件命名噩梦，那么你来对地方了。Git 不是选修课，它是每一个写代码的人必须掌握的基础设施。本教程用 6 章带你从零到熟练，不多废话。
:::

## 适合谁

- 计算机相关专业的大学生，希望系统掌握版本控制
- 正在准备实习或求职的准程序员，Git 是面试和工作的硬门槛
- 对开源和 GitHub 感兴趣，想参与实际项目的开发者

## 你将学到什么

1. **理解版本控制** —— 为什么它是程序员的第一堂工程课
2. **掌握核心操作** —— `add`、`commit`、`push` 等日常指令，练到肌肉记忆
3. **学会分支管理** —— 并行开发、合并冲突，多线作战不乱
4. **连接远程仓库** —— 将代码托管到 GitHub，开始你的协作之旅
