# 命令行是什么——理解黑客的自我修养

::: tip 写在前面
如果在黑窗口前手足无措，你来对地方了。命令行不是黑客电影里的特效背景，它是每一个程序员与计算机对话的最直接方式。本教程用 8 章带你从零到熟练，不多废话。
:::

## 适合谁

- 计算机相关专业的大学生，希望系统掌握命令行操作
- 正在准备实习或求职的准程序员，命令行是日常工作的基础环境
- 对 Linux、运维、后端开发感兴趣，想打好地基的开发者

## 你将学到什么

1. **理解命令行的本质** —— 它不是复古工具，而是与计算机对话的另一种语言
2. **掌握基础操作** —— 浏览文件、创建目录、运行程序，像使用文件管理器一样自如
3. **认识文件系统** —— 绝对路径与相对路径、权限模型，命令行下的文件世界
4. **学会管道与重定向** —— 理解 Unix 哲学："每个程序只做一件事，并把事情做好"
5. **了解不同的 Shell** —— CMD、PowerShell、Bash 的前世今生与各自战场
6. **搞懂环境变量** —— 困扰无数新手的 PATH，其实是一个朴素的设计
7. **掌握 SSH 远程连接** —— 一行命令登录另一台电脑，这是程序员的基本功
8. **入门 Shell 脚本** —— 把重复操作自动化，让计算机替你打工

## 学习建议

::: info 预计学习时间
- 快速浏览：2-3 小时
- 边学边练：5-8 小时
- 深入探索：按需延伸
:::

::: warning 学习心态
命令行学习和学骑自行车很像——一开始可能会摔几跤（敲错命令、找不到文件），但一旦掌握了，你就再也不会忘记。关键是**动手实践**，光看不练是学不会的。
:::
