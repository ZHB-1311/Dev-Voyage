# 别慌，计算机其实很好懂

> 手机掉水里了。买新的，下载微信，登录——聊天记录全没了。登录淘宝，购物车空了。登录游戏，进度归零。
>
> 电脑 C 盘红了，桌面堆满文件，每次开机像在等火车。想清理又不敢乱删。
>
> Wi-Fi 显示已连接，但网页打不开。你重启了路由器，好了五分钟又不行。
>
> 这些东西在学校没人教，出了问题只能到处求人。这门课就是来填这个坑的。

如果你符合以下任何一条：

- 手机玩得飞起，打开电脑连文件都找不到
- 每次装软件都小心翼翼，怕点什么奇怪的东西
- 不明白为什么同一个 Wi-Fi，手机能连但电脑不行
- 看到"系统更新"弹窗就纠结要不要点
- 觉得命令行是黑客才用的东西

这门课就是写给你的。不需要任何基础——我们假设你只有手机使用经验，但想真正理解这些每天都在用的东西到底是怎么工作的。

## 学完你能干什么

- **电脑卡 / Wi-Fi 断 / 文件找不到**——知道怎么排查，不用次次求人
- **几十个账号密码**——知道怎么管，不再靠"全部设为同一个密码"
- **想学编程 / 剪视频 / 做设计**——这套常识是地基，没有它你后面会一直卡
- **被推荐算法喂得死死的**——理解它的逻辑，反过来控制它

## FAQ

**Q：我真的零基础，能学会吗？**

A：这门课就是为纯小白设计的。遇到不懂的，往下看，答案大概率在后面。

**Q：需要什么设备？**

A：有手机和一台电脑就够了。没有电脑？手机也能学懂大部分概念。

**Q：学完能当程序员吗？**

A：不能。但如果你想学编程，这是最好的起点——我们打的是地基，不是装修。

**Q：为什么不用专业术语？**

A：术语是学习的结果，不是起点。先理解概念，术语自然就来了。
