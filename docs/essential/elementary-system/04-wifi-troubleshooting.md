# 第 4 章：Wi-Fi 连不上？先试试"重启大法"

> 手机刷抖音流畅无比，电脑开腾讯会议卡成 PPT。同一个 Wi-Fi，同一个房间，体验天差地别。
>
> 你重启了电脑，不行。重启路由器，好了五分钟又不行。最后发现——隔壁在下载东西。
>
> 网络问题 90% 的情况下重启路由器就能解决。剩下 10%，才是真正值得搞明白的东西。

## 网络翻车现场

小美在家上网课：手机刷抖音流畅，电脑上腾讯会议卡成 PPT。重启电脑不行，重启路由器好了五分钟又不行。最后发现——不是设备的问题，是有人在占带宽。

网络故障的原因往往不在你这边，但排查得从你能控制的步骤开始。

## Wi-Fi 是怎么工作的

简化到最本质的模型，连接过程就四步：

1. **路由器广播信号**："我叫 TP-Link_xxxx，密码是 xxxxxxxx"
2. **你的设备响应**："我要连接，密码对上了"
3. **路由器分配 IP 地址**："好，你坐 192.168.1.101"
4. **连接建立**

大部分网络问题出在步骤 3（IP 分配冲突）或步骤 1（信号太弱/干扰太多）。

### 2.4GHz 和 5GHz

现在的路由器基本都是双频的，两个频段的区别很直接：

::: info 2.4G vs 5G：怎么选
|  | 2.4GHz | 5GHz |
|:---|:---|:---|
| **速度** | 慢 | 快 |
| **穿墙能力** | 强 | 弱（遇墙明显衰减） |
| **干扰** | 多（蓝牙、微波炉都在用这个频段） | 少 |
| **适合场景** | 离路由器远、隔墙多 | 离路由器近、打游戏、看 4K 视频 |

**基本原则：能连 5G 就连 5G，信号不好再切回 2.4G。**
:::

如何判断你的设备支持哪个？2015 年后的手机/电脑基本都支持 5GHz。看路由器——如果有两个 Wi-Fi 名称（其中一个带 `_5G` 后缀），就是双频路由器。

实际操作：
- 打游戏、看高清视频 → 连 5GHz
- 离路由器远、隔墙多 → 连 2.4GHz
- 智能家居设备（扫地机器人、智能灯）→ 通常只支持 2.4GHz

## 为什么手机能连，电脑不行

这是最让人抓狂的场景。按概率排序：

**第一名：IP 地址冲突（约 40%）**
- 症状：显示"已连接，无 Internet"
- 原因：两台设备被分配了同一个 IP 地址
- 解决：重启路由器或电脑，重新分配 IP

**第二名：网卡驱动过时（约 30%）**
- 症状：搜不到 Wi-Fi，或信号满格但速度极慢
- 原因：电脑的 Wi-Fi 模块驱动版本太老，跟新的路由器协议不兼容
- 解决：更新网卡驱动

**第三名：电源管理设置（约 20%）**
- 症状：用着用着突然断线，过会儿又自动连上
- 原因：Windows 为了省电让网卡间歇性"睡眠"
- 解决：关闭网卡的电源管理模式

**第四名：路由器限制（约 10%）**
- 症状：特定设备永远连不上
- 原因：路由器设置了 MAC 地址过滤
- 解决：登录路由器后台，把设备加入白名单

### 诊断流程

```
电脑连不上 Wi-Fi
  ↓
第一步：重启路由器（拔电源等 30 秒）
  ↓
还不行 → 第二步：重启电脑
  ↓
还不行 → 第三步：忘记网络，重新连接
  ↓
还不行 → 第四步：更新网卡驱动
  ↓
还不行 → 第五步：检查 IP 设置（设为自动获取）
  ↓
还不行 → 可能是硬件问题，考虑外接 USB 网卡
```

::: tip 为什么重启大法永远有效
路由器本质上是一台小电脑，有 CPU、内存、操作系统。运行久了会积累问题：内存泄漏、缓存溢出、ARP 表撑爆——重启就是把所有状态清空，从头来过。

拔电源等 30 秒不是因为玄学，而是为了让主板上的电容器彻底放电，确保内存里的残留数据全部清空。
:::

## 各系统操作指南

### Windows

**查看网络状态：**
右下角网络图标 → 右键 → "打开网络和 Internet 设置" → "更改适配器选项" → 右键"WLAN" → 状态。看"速度"一栏，72Mbps 以上算正常。

**更新网卡驱动：**
右键开始菜单 → 设备管理器 → 展开"网络适配器" → 右键你的无线网卡（名字通常带 Wireless 或 Wi-Fi） → "更新驱动程序" → "自动搜索"

**关闭网卡省电模式（解决频繁断连）：**
设备管理器 → 右键无线网卡 → 属性 → "电源管理"选项卡 → 取消勾选"允许计算机关闭此设备以节约电源"

**网络重置（终极大招）：**
设置 → 网络和 Internet → 状态 → 网络重置

::: caution 网络重置会清除所有 Wi-Fi 密码和网络配置
相当于从头设置网络。慎用，但确实有效。
:::

### macOS

**查看网络详情：**
按住 Option 键点击菜单栏 Wi-Fi 图标，能看到信号强度（RSSI）、信道、传输速率等详细信息。

**删除网络配置重新连接：**
系统设置 → Wi-Fi → 高级 → 找到有问题的网络 → 点"-"删除 → 重新搜索连接

**终端命令修复：**
```bash
sudo ipconfig set en0 DHCP      # 重新获取 IP
sudo dscacheutil -flushcache    # 刷新 DNS 缓存
```

### 安卓 / iOS

**安卓：**
设置 → 网络和 Internet → Wi-Fi → 长按连接的 Wi-Fi → 修改网络 → IP 设置 → 从 DHCP 改为静态，再改回 DHCP

**iOS：**
设置 → Wi-Fi → 点击当前 Wi-Fi 后面的"i" → 点击"忘记此网络" → 重新连接

**临时解决方案：**
电脑连不上但手机可以时，手机开个人热点 → 电脑连手机热点上网 → 趁能上网查解决方案或下载驱动

## 路由器设置

你不需要成为网络工程师。会这三步就够了。

**第一步：找到路由器 IP**
- Windows：`Win + R` → 输入 `cmd` → 输入 `ipconfig`，看"默认网关"
- macOS：系统设置 → 网络 → 高级 → TCP/IP，看"路由器"
- 通常是：`192.168.1.1`、`192.168.0.1`、`192.168.50.1`

**第二步：登录后台**
浏览器输入路由器 IP → 输入用户名密码（通常在路由器底部贴纸上）。常见组合：admin/admin、admin/123456。

**第三步：关键设置检查**
1. Wi-Fi 名称和密码——在这里改
2. 信道——2.4GHz 选 1、6、11（干扰最少）；5GHz 选 149 以上
3. 频宽——2.4GHz 选 20MHz；5GHz 选 80MHz
4. DHCP 服务器——确保开启，IP 池范围够大

::: warning 这几个设置不要乱碰
- WAN 口设置——改错了直接上不了网
- 运营商拨号信息——记下原来的再改
- 恢复出厂设置——除非你知道自己在干什么
:::

### 路由器摆放

Wi-Fi 信号是电磁波，遇金属和混凝土大幅衰减。放对位置比买贵的路由器更重要。

**正确位置：**
- 房子中央，高度 1.5 米左右（桌面或柜子上）
- 远离金属物体（冰箱、微波炉）
- 远离承重墙和镜子
- 天线竖直向上（信号水平扩散最佳）

**容易被忽略的干扰源：台灯。**

可见光不影响 Wi-Fi。但劣质台灯的电源适配器（特别是没有 3C 认证的便宜货）会产生低频谐波，正好落在 2.4GHz 频段附近。如果你发现路由器旁边的台灯一开 Wi-Fi 就变慢，换个台灯比换路由器更管用。

**错误位置：**
- 塞在弱电箱里（金属门屏蔽信号）
- 放在地上（信号被家具遮挡）
- 藏在电视后面（电磁干扰）

**信号改善方案：**
- 单间信号差 → 调整位置，或买信号放大器（中继器）
- 大户型/多楼层 → Mesh 组网（多台路由器协同）
- 无法布线且穿墙差 → 电力猫（通过电线传输网络）

## 校园网特别篇

宿舍的网络环境和家庭完全不同，有自己的特殊问题。

### 认证与限制

校园网通常使用 Dr.COM、锐捷（Ruijie）或天翼客户端进行认证——连接网线或 Wi-Fi 后打开浏览器，输入学号密码才能上网。

常见限制：
- **一个账号只能登录 2-3 台设备**——手机、电脑、平板，多了就被踢
- **MAC 地址绑定**——第一次认证时绑定了你的设备，换电脑或手机需要解绑
- **限速**——晚高峰时段带宽被均分，打游戏卡
- **断电断网**——有些宿舍晚上 11 点后断网或强制断网

### 让路由器在宿舍工作

想把校园网分给多个设备用，需要一台路由器，但设置比家庭复杂：

1. 把校园网网线插到路由器的 **WAN 口**
2. 进入路由器后台，上网方式选 **动态 IP** 或 **锐捷认证**（部分路由器支持，如小米、TP-Link 的某些型号）
3. 在路由器里登录你的校园网账号
4. 所有设备连路由器的 Wi-Fi 就能上网，只占一个账号名额

如果路由器不支持认证协议，另一个方案是**用电脑开热点**——Windows/macOS 都可以把有线网络通过 Wi-Fi 共享出来。缺点是电脑不能关。

### 设备超限被踢怎么办

如果账号被踢下线：
- **路由器方案：** 在路由器后台查看已连接设备，关掉不用的
- **电脑方案：** 断开不用的设备，在校园网自助页面解绑多余的 MAC
- **校园网客户端**可能限制了共享上网，需要开 MAC 地址克隆（路由器后台一般有这个功能）

## 4G/5G 移动网络

### 信号满格但没网？

三个可能原因：
1. **基站拥堵**——演唱会、晚高峰时段的经典问题
2. **信号虚标**——手机显示的是信号强度，不是网络质量
3. **运营商限速**——所谓"不限量套餐"超过一定流量后被降速

**改善方法：** 开启飞行模式 10 秒再关闭（强制重新连接基站）。有条件的话切换到 3G 试试——有时候 4G 堵了 3G 反而能用。真不行就打运营商电话投诉，确实有用。

### 5G 的真实体验

5G 的理论优势很明显：低延迟（打游戏几乎无感）、高速度（理论比 4G 快 10-100 倍）、大连接（物联网基础）。

但 5G 的问题也很直接：耗电快（手机续航明显下降）、覆盖差（基站密度需要是 4G 的 3-5 倍）、穿透弱（进室内信号衰减严重）。

残酷的现实：很多人开着 5G 套餐，实际速度和 4G 差不多——基站还没建好、运营商偷偷限速、手机发热自动降频。不是 5G 不行，是基础设施还没跟上。

## 公共 Wi-Fi 安全

公共 Wi-Fi 的三个主要风险：
- **中间人攻击：** 黑客伪装成 Wi-Fi 热点，截获你的所有流量
- **嗅探：** 同一网络下其他人可以看到你在传什么
- **恶意热点：** "麦当劳_Free" 可能是黑客设的陷阱

**绝对不要在公共 Wi-Fi 上做的事：**
- 登录网银、支付宝、微信支付
- 输入信用卡信息
- 登录重要账号（邮箱、社交账号）

**如果不得不连：**
1. 向店员确认正确的 Wi-Fi 名称
2. 使用 VPN 加密所有流量
3. 开启手机的"随机 MAC 地址"功能
4. 用完立即"忘记此网络"

::: warning 免费 VPN 反而更危险
很多免费 VPN 偷偷卖你的数据、植入恶意软件、速度慢到没法用。如果确实需要 VPN，选知名付费品牌。
:::

### "万能 Wi-Fi 钥匙"是陷阱

这类软件号称"一键破解 Wi-Fi 密码"，但原理跟破解没关系——你连上某个 Wi-Fi 后，它把密码上传到云端数据库。别人在附近也能查这个数据库连上来。

也就是说：你用它的同时，**你家路由器的密码也可能已经被共享出去了。**

所以：
- 它"破解"不了你不知道密码的 Wi-Fi——只能查别人已经分享过的
- 你自家 Wi-Fi 的密码，可能已经在数据库里裸奔了
- 这类 APP 本身就是最大的隐私风险

## 常见问题

**Q：晚上 8-11 点网速特别慢？**

A：这是晚高峰，整栋楼都在上网。切到 5GHz 频段（用的人少），或者联系运营商升级带宽。大文件下载可以设置凌晨自动进行。

---

**Q：显示已连接但打不开网页？**

A：DNS 问题。网络设置里把 DNS 改成 `8.8.8.8` 和 `114.114.114.114` 试试。

---

**Q：路由器多久重启一次？**

A：普通家用每周一次。多人合租每 3 天一次。打游戏的话每次玩之前重启一次。大部分路由器后台可以设置定时重启。

---

**Q：Wi-Fi 辐射有害吗？**

A：Wi-Fi 辐射功率约 0.1 瓦，手机通话时约 1-2 瓦。阳光的辐射功率约 1000 瓦/平方米。**Wi-Fi 的辐射强度不到阳光的万分之一。**

---

**Q：怎么查谁连了我的 Wi-Fi？**

A：登录路由器后台 → 找"设备列表"或"已连接设备"。发现陌生设备就改密码。

---

**Q：Wi-Fi 密码忘了怎么办？**

A：如果电脑已连过，在网络连接里查看属性并勾选"显示字符"。路由器底部贴纸上有默认密码。或者登录路由器后台查看。都不行就只能重置路由器了。

---

**Q：想学自己压网线，值得吗？**

A：买一把压线钳（20-30 块）、一盒水晶头、一卷网线，自己做一根的成本不到成品的三分之一，而且想要多长做多长。看着很专业，实际三分钟学会——网上搜"网线线序 T568B"就有教程。宿舍里自己拉一根长网线到床位，比靠 Wi-Fi 穿墙稳多了。

## 局域网和互联网

你连上 Wi-Fi 后，设备会获得一个类似 `192.168.x.x` 的 IP 地址——这是**局域网 IP**（内网 IP），只在你的路由器管辖范围内有效。你的设备通过路由器访问互联网，外界不能直接连进来。

这个结构解释了三个常见问题：

**为什么局域网传文件比百度网盘快得多？**
千兆局域网理论速度约 125MB/s，你家宽带下载速度一般只有 10-50MB/s。同在一个 Wi-Fi 下的设备传文件走的是局域网，不经过互联网——所以手机往电脑传大文件，用 AirDrop 或 SMB 共享比上传到网盘再下载快得多。

**为什么打游戏插网线比 Wi-Fi 稳？**
Wi-Fi 有抖动和丢包，即使延迟很低，那 1% 的丢包率在 MOBA/FPS 游戏里就是跳 ping 和瞬移。网线虽然丑但稳定——有线连接没有无线干扰，延迟波动小得多。

**什么是"内网穿透"？**
因为外界不能直接访问你的内网设备，如果你想在外面访问家里的电脑（远程桌面、NAS），就需要内网穿透——把内网的服务"映射"到公网上。这就是 frp、ZeroTier、Tailscale 这类工具做的事情。

::: info Wi-Fi 背后的冷知识
Wi-Fi 这个名字没有任何含义。它不是"Wireless Fidelity"的缩写，只是一个品牌名——当时 Wi-Fi 联盟想要一个比"IEEE 802.11b Direct Sequence"更亲民的名字，品牌咨询公司提供了 10 个选项，他们挑了这个听起来像 Hi-Fi 的。

更值得知道的是 Wi-Fi 的核心技术来源。跳频扩频（Frequency Hopping Spread Spectrum）——Wi-Fi 和蓝牙的基础——是一位好莱坞女演员发明的。她叫海蒂·拉玛，二战期间为了防止鱼雷无线电信号被干扰，从钢琴演奏中获得灵感，设计了一种让频率像琴键一样不断跳变的技术。这项专利在 1942 年注册，几十年后成了 CDMA、蓝牙和 Wi-Fi 的基石。

所以每当你说"Wi-Fi 信号不好"的时候，你其实是在抱怨一位好莱坞女演员在 80 年前设计的方案还不够完美。
:::
