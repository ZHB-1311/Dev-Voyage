# 第 7 章：多设备协同：让手机和电脑谈恋爱

> 手机收到 10 张产品照片要在电脑上用。用微信传——画质被压缩、顺序全乱了。U 盘没带，数据线没带，邮箱附件太大。
>
> 最后他用手机对着电脑屏幕一张张拍。
>
> 每个人都经历过这种"设备孤岛"时刻。其实至少有 5 种更优雅的解法。

## 从穿球鞋的网络说起

1980 年代传大文件最快的方法是把文件拷进软盘，跑着送到隔壁——网速只有几 KB/s 的电子邮件远不如跑腿快。这被戏称为 Sneakernet（穿球鞋网络）。

直到今天，谷歌传输 PB 级数据时依然用卡车拉着硬盘在公路上跑——因为这比光纤还快。有些问题的解决方案，几十年都没变过。

## 文件传输

### 为什么微信文件传输不好用

画质压缩（10MB → 200KB）、大小限制（超过 1GB 传不了）、文件散落在聊天记录里过期被清理。微信的设计目标是聊天，不是文件管理。

### 国内生态互联

国内手机厂商这两年都在做自家生态，对大学生来说可能比苹果方案更实际——因为你的手机、平板、笔记本大概率是同一个牌子。

**华为：鸿蒙生态 + 华为分享**
鸿蒙的核心能力是"分布式"——多个设备可以组合成一台"超级终端"。手机、平板、笔记本在控制中心里拖到一起就能互传文件、共享剪贴板、任务接续。文件直接在电脑上打开编辑，手机后台编辑的文件电脑上同步更新。

硬件联动：手机和耳机靠近自动弹窗配对，平板可以作为电脑的第二块屏幕，手表可以控制手机拍照。鸿蒙座舱已搭载在问界、智界等车型上——手机导航上车自动流转到车机，车上没听完的歌下车回到手机上继续。

**小米：澎湃 OS + HyperConnect**
小米的 HyperConnect 覆盖手机、平板、笔记本、智能家居。核心功能：跨设备剪贴板同步、手机文件直接拖到电脑、手机相机当电脑摄像头用、平板当电脑扩展屏。平板端可以接力手机上没看完的视频和文档。小米 SU7 的车机同样接入澎湃生态——手机上的导航和音乐上车自动接管，车到家门口地库自动联动回家模式（灯、空调、加湿器）。

**小米：澎湃 OS + HyperConnect**
小米的 HyperConnect 覆盖手机、平板、笔记本、智能家居。核心功能：跨设备剪贴板同步、手机文件直接拖到电脑、手机相机当电脑摄像头用、平板当电脑扩展屏。平板端可以接力手机上没看完的视频和文档。

**vivo / OPPO：办公套件 + 跨屏互联**
两家各自的 PC 客户端（vivo 办公套件、OPPO 跨屏互联）都支持：手机投屏到电脑、文件拖拽、剪贴板共享、通知同步。算是稳健但没那么花哨的生态。

**互传联盟**
小米、OPPO、vivo 组成了互传联盟，不同品牌手机之间可以像 AirDrop 一样快速传文件。这是一个很实际的联盟——不考虑你的室友用什么牌子的手机，都能直接传。

### 同生态的官方方案（国际）

**苹果：AirDrop（隔空投送）**
Wi-Fi + 蓝牙点对点传输，最快约 40MB/s，不需要网络。手机打开照片 → 选择 → 点分享按钮 → 在分享面板找到电脑用户名 → 电脑点"接受"。

隐藏技巧：按住 Shift 键点菜单栏 Wi-Fi 图标可查看 AirDrop 状态；可以设为"仅限联系人"或"所有人"；能传任何文件，不只是照片。

**华为：华为分享 / 超级终端**
类似 AirDrop 但更深度——多屏协同让电脑直接操作手机界面，文件拖拽到电脑文件夹，任务接续。手机下拉控制中心开启"华为分享"，电脑打开"华为电脑管家"，手机靠近电脑按提示连接。

**小米 / OPPO / vivo 等：互传联盟**
国内安卓厂商组成了互传联盟，不同品牌间快速传文件。与电脑的互联体验不如华为。

**微软：Phone Link（手机连接）**
Windows 搜索"手机连接"→ 选择 Android → 扫码配对 → 手机上装 Link to Windows。可查看通知、收发短信、传输照片。在电脑上使用手机 APP 仅限三星部分机型。

### 跨生态方案

iPhone 遇到 Windows，或者 Android 遇到 Mac：

**Snapdrop / PairDrop（零安装）：** 打开网页就能互传文件，全平台通杀。适合临时场景——你和室友各用不同设备，不需要装任何软件。

**Feem：** 全平台，局域网传输。

**LANDrop：** 开源免费，无大小限制。

**云盘中转：** 手机照片自动备份到网盘，电脑登录同一个账号下载。不能即时，但不担心漏文件。

**局域网共享：** 手机和电脑连同一个 Wi-Fi，电脑创建共享文件夹，手机用文件管理器访问 `\\电脑IP地址`。Windows 需开启"网络发现"和"文件共享"。全平台通用，没有大小限制。

## 任务接续

经典场景：地铁上用手机看文章，到办公室打开电脑，点一下继续看。

**苹果：接力（Handoff）**
支持 Safari 网页、邮件、地图、备忘录、Pages 等。设置 → 通用 → 隔空播放与接力 → 开启"接力"。到 Mac 前 Dock 栏自动出现来自手机的图标。

**微软：时间线（Timeline）**
Edge 浏览器和 Office 文档的活动记录在设备间同步。Win+Tab 查看时间线。

**谷歌：Chrome 同步**
手机和电脑都用 Chrome 并登录同一账号。电脑 Chrome → 历史记录 → 标签页 → 来自其他设备。

体验概括：苹果最无缝（Dock 栏直接出现），谷歌最通用（任何设备登录 Chrome 就行），微软在中间（限 Edge 生态）。

## 通知同步与剪贴板同步

### 通知同步

**苹果：** 手机来电时 Mac/iPad 也会响铃，可以直接接听。短信在所有设备可见可回。设置：iPhone → 电话 → 在其他设备上通话。

**微软 Phone Link：** 在电脑上看到手机通知（微信、短信、来电），可以回复短信。接电话需要蓝牙耳机。

**KDE Connect：** 开源免费，支持 Linux/Windows/macOS/Android。一个 APP 覆盖通知同步、剪贴板共享、文件传输、远程输入、媒体控制。如果你不用苹果生态，这是最强的免费多设备方案。

### 剪贴板同步

痛点：手机上复制了一段文字，想在电脑上用。

**苹果通用剪贴板：** iPhone 上复制，Mac 上直接粘贴。要求同一 Apple ID，开启蓝牙和 Wi-Fi。

**KDE Connect：** 同上，跨平台跨品牌。

**微软 SwiftKey / Clipbrd：** 输入法内置剪贴板同步。

**最粗暴的方案：** 微信/QQ 发给自己。虽然 low，但有效。

## 投屏方案

把手机画面投到电脑或电视上。

**Miracast（Windows 原生）：** 不需要网络，点对点投屏。Win+K 打开连接面板，手机选"无线显示"。延迟较高，看视频可以，打游戏不行。

**AirPlay（苹果）：** iPhone/iPad 投到 Mac 或 Apple TV。控制中心 → 屏幕镜像。体验最流畅，限苹果设备。

**Google Cast（安卓）：** 手机投到 Chromecast 或支持 Cast 的电视。视频 APP 基本都内置 Cast 按钮。

**第三方方案：** AirDroid Cast 投屏、Spacedesk 把平板当扩展屏幕。

## 双显示器设置

宿舍外接显示器是常见场景。一个屏幕写代码、一个屏幕看文档。

**Win+P：** 选扩展模式。主显示器设为你面朝的那块（设置 → 系统 → 显示 → 选主显示器）。

**macOS：** 系统设置 → 显示器 → 排列。白色菜单栏所在的是主显示器。

**快捷键：** Win+Shift+方向键移到另一个屏幕，Win+方向键贴边。macOS 推荐 Rectangle（免费）实现类似功能。

## 远程控制

出门了，电脑没关，需要取个文件。

**Chrome 远程桌面：** 最简单。访问 remotedesktop.google.com 设置。只需要 Chrome 浏览器。

**Windows 远程桌面：** 免费，但需要电脑有公网 IP，或做内网穿透——对宿舍网络不适用。

**TeamViewer / AnyDesk：** 个人免费，手机和电脑都装客户端，用 ID 和密码连接。

**苹果屏幕共享：** 限 Mac 控制 Mac。设置 → 共享 → 屏幕共享，另一台 Mac 的访达侧边栏找到这台电脑。

::: tip 内网穿透的现代方案：Tailscale / ZeroTier
宿舍网络和校园网一般没有公网 IP，Windows 远程桌面用不了。Tailscale 和 ZeroTier 可以在所有设备之间建立一个加密的虚拟网络——设备间可以直接互相访问，不需要公网 IP，不需要在路由器上做端口转发。两台设备都装上 Tailscale 登录同一个账号，就能像在同一个局域网一样互相访问。
:::

**帮父母解决手机问题：**
小米/华为等品牌自带"远程协助"（需同品牌手机）。AirDroid 功能更全但设置复杂。最简单的方案：微信视频时开启屏幕共享——可以看到对方屏幕，口头指导，不能直接操作，但大部分情况够用。

## 传输技术演进

回顾一下手机之间传文件的方式是怎么进化的，能帮你理解现在的方案为什么这样设计。

**红外传输（1998-2005）：** 两个红外口对准，不能动，不能有遮挡。速度约 14KB/s，传一张照片要 30-60 秒。经典机型：诺基亚 8310。

**蓝牙（2000 年至今）：** 不用对准了，但配对繁琐，速度慢。经典场景：传铃声、传小说、联机打贪吃蛇。蓝牙 5.0 的速度已经提升到 2 Mbps，主要优势是低功耗。

**Wi-Fi Direct（2010 年）：** 设备间直接建立 Wi-Fi 连接，不经过路由器。速度可达 250 Mbps。这是 AirDrop、华为分享等现代方案的底层技术。

**UWB 超宽带（2019 年起）：** iPhone 11 首次引入。用于 AirTag 精确定位、数字车钥匙。未来设备靠近自动连接。

|:---|:---|:---|
| **iPhone + Mac** | 苹果全家桶 | AirDrop、接力、通用剪贴板、通话同步 |
| **华为手机 + 华为笔记本** | 鸿蒙生态 + 华为分享 | 超级终端、多屏协同、文件拖拽、剪贴板同步 |
| **小米手机 + 小米笔记本** | 澎湃 OS + HyperConnect | 跨端剪贴板、手机当摄像头、平板扩展屏 |
| **vivo/OPPO + Windows** | 办公套件 / 跨屏互联 | 投屏、文件拖拽、通知同步 |
| **安卓 + Windows（非同一品牌）** | Phone Link + KDE Connect | 通知同步、剪贴板共享、文件传输 |
| **混搭组合** | Snapdrop + KDE Connect + Tailscale | 零安装传文件、多设备同步、远程访问 |
| **全平台** | 云服务 + Tailscale | iCloud/OneDrive/Google Drive + 设备互联 |

### 30 分钟搭建指南

**10 分钟：** 确保所有设备登录同一品牌账号，开启 Wi-Fi 和蓝牙，更新系统。
**10 分钟：** 苹果设隔空投送为"联系人"；华为开启华为分享；其他安装 KDE Connect。
**10 分钟：** 设置默认浏览器并登录账号，开启剪贴板同步，安装 Tailscale。

### 避坑指南

**AirDrop 找不到对方设备？** 检查 Wi-Fi 和蓝牙是否开启，隔空投送是否设为"所有人"（2012 年前设备可能不支持），重启 Wi-Fi 和蓝牙。

**多屏协同延迟高？** 确保手机和电脑在同一个 5GHz Wi-Fi 网络，关闭其他设备的高带宽占用，靠近路由器。

**远程控制连接不上？** 检查防火墙是否允许远程桌面，确认是否在同一个网络或有公网 IP（用 Tailscale 可绕过此问题），检查账号密码是否正确。

## 未来展望：智能家居

**理想：** 回家自动开灯开空调，出门自动关电器。
**现实：** 不同品牌的电器需要不同的 APP，联动设置复杂。
**现状：** 小米米家、苹果 HomeKit、华为 HiLink 各立山头。

## 不止手机和电脑：汽车与智能家居

多设备协同已经不局限于手机、平板、笔记本。汽车和家庭设备也在接入这张网。

### 手机就是车钥匙

宝马、特斯拉、比亚迪等已经支持手机解锁。走近自动解锁，离开自动上锁，手机没电也能用备用电量支撑。小米 SU7 更是把手机生态直接搬进了车机——上车自动连接，手机上的导航、音乐无缝流转到车机屏幕。华为鸿蒙座舱也是同样的逻辑：手机和车机登录同一华为账号，手机上正在听的歌、看的导航路线，上车自动接管。

### 智能家居：从单品到全屋

"回家自动开灯、空调调到 26 度、扫地机器人开始工作"——这些不再是科幻。小米米家是目前国内最完整的智能家居生态：灯具、空调、窗帘、门锁、传感器、电视、空气净化器，几百款设备同一个 APP 控制。华为 HiLink 覆盖类似，优势是路由器和摄像头。苹果 HomeKit 设备少但隐私好。

### 互联的未来：Matter

Matter 协议（2022 年发布 1.0）由苹果、谷歌、亚马逊联合制定，目标是让不同品牌的智能设备互联互通。如果 Matter 普及，以后买智能灯不再需要问"支持米家还是 HomeKit"——只要支持 Matter 就行。目前 Matter 设备在逐渐增多，距离全面普及还需要几年。

### 一个账号贯通所有设备

你的手机、电脑、平板、手表、汽车、台灯、空调、门锁……都登录同一个账号。这就是生态的终极形态。好处是方便到难以想象——坏处是你换阵营的难度也高到无法承受。

## 常见问题

**Q：多设备同步会耗电吗？**

A：会，但不多。蓝牙和 Wi-Fi 待机功耗很低。耗电明显增加通常是某个设备在频繁尝试连接失败。

---

**Q：不同品牌的设备能深度互联吗？**

A：不能。苹果、华为、小米都在打造自己的生态围墙。跨品牌只能用第三方工具或云服务，体验打折扣。

---

**Q：公司电脑能设置多设备协同吗？**

A：谨慎。公司电脑可能有安全策略限制，私自安装软件或开启远程访问可能违反规定。先咨询 IT 部门。

---

**Q：AirDrop 安全吗？会被陌生人截获吗？**

A：AirDrop 使用加密传输。公共场所设为"仅限联系人"，不要随意接受陌生人的文件，传完敏感文件后关闭。

---

**Q：Windows 电脑能用 AirDrop 吗？**

A：原生不支持。替代方案：Snapdrop（网页）、LANDrop、Feem、数据线。

---

**Q：旧设备支持这些功能吗？**

A：AirDrop 需 2012 年后苹果设备，华为多屏协同需 EMUI 10+，Phone Link 需 Win 10 2019 年 5 月更新及以上。KDE Connect 和 Tailscale 对旧设备兼容性最好。

1960 年代，大型计算机一台几百万美元。为了让更多人能使用，研究人员发明了"远程终端"——一个只有显示器和键盘的设备，通过电话线连接到主机。

这个远程终端就是现代远程桌面的雏形。当你用手机控制宿舍的电脑时，本质上用的是 60 年前的技术——只不过从电话线变成了互联网，从大型机变成了个人电脑。
