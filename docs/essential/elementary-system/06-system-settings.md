# 第 6 章：设置里的迷宫：这些开关到底开不开？

> 手机弹出了系统更新提示。点开更新说明，写着"错误修复和安全性改进"。上网搜了一圈——有人说"更新后续航崩了"，有人说"不更新会被黑客攻击"。
>
> 一周过去，那个红点还在。

## 操作系统的启动之旅

在你看到桌面之前，你的电脑已经执行了几十步操作。理解这个过程能帮你理解很多"为什么电脑要这样设计"。

```
按下电源键
  ↓
BIOS/UEFI 自检（POST）——检查硬件是否正常
  ↓
引导加载程序 —— 找到硬盘上的操作系统
  ↓
加载内核 —— 操作系统的核心启动
  ↓
启动系统服务 —— 网络、声音、图形界面…
  ↓
显示登录界面
```

### BIOS 和 UEFI

这是电脑通电后运行的第一个程序。它不装在硬盘上，而是写在主板的一个芯片里。

**怎么进：** 开机时按特定键——通常是 F2、Del、F12（不同主板不同，开机画面会闪一下提示）。进不去的话去搜"你的电脑型号 + 进 BIOS"。

**在里面能干什么：**
- 改启动顺序（从 U 盘启动装系统）
- 开启虚拟化（VT-x / AMD-V 虚拟化技术——跑 WSL2、Android 模拟器、VMware 都需要这个）
- 调内存频率（XMP/EXPO——让内存跑在标称速度而非默认低速）

**别乱动什么：**
- CPU 电压和频率（稳定性灾难）
- 启动顺序改完了记得改回来，不然拔了 U 盘开不了机
- 安全启动（Secure Boot）——关掉才能从某些 U 盘启动，但关了会降低系统安全性

::: info Windows vs Linux vs macOS 的启动差异
**Windows** 使用 NT 内核，启动过程依赖 Boot Manager，启动后加载大量系统服务（所以开机慢，但功能全）。

**Linux** 内核启动后基本是"你有什么硬件我加载什么驱动"，启动过程简洁，所以 Linux 开机通常比 Windows 快。多数 Linux 发行版使用 GRUB 作为引导加载程序（就是开机时让你选"Ubuntu / Windows"的那个菜单）。

**macOS** 使用 XNU 内核（混合内核），启动过程有 T2 芯片或 Apple Silicon 的安全协处理器参与验证——每个启动阶段都要签名验证，所以越旧的 Mac 装新系统越困难。
:::

## 新电脑到手第一件事

大一新生拿到新电脑，别急着联网激活。先做这三件事：

**1. 跳过联网激活（Windows 11）**
Windows 11 强制要求登录微软账号才能继续。在联网界面按 `Shift + F10`，输入 `oobe\bypassnro`，电脑重启后会出现"我没有 Internet 连接"的选项——点它，就能创建本地账号。

**2. 卸载预装软件**
电脑厂商预装的杀毒软件试用版、PDF 阅读器、音乐播放器…大部分你不需要。去设置 → 应用 → 应用和功能，把不认识的全看了，不需要的卸掉。

**3. 关掉不必要的开机自启**
任务管理器 → 启动，禁用除了硬件驱动和输入法之外的所有东西。

::: tip Windows 可以不激活
微软不强制激活 Windows。不激活只是右下角有水印+无法个性化（改壁纸、主题色）。对于预算有限的学生，这比去下"激活工具"安全得多——那些激活工具本质上是特洛伊木马。

如果你实在需要激活，正规渠道是去淘宝买序列号（几十块），不要下载任何"KMS 激活工具"。
:::

## 系统更新

（以下内容保留原有更新策略）

一个典型的系统更新包含：

- 安全性更新——修复漏洞，**应该尽快安装**
- 错误修复——解决崩溃和异常
- 性能改进——优化电池、流畅度
- 功能更新——新功能，**可以等一个月看看反馈**

| 系统 | 建议策略 |
|:---|:---|
| **Windows** | 安全更新立即装，大版本等 1 个月 |
| **macOS** | 小更新随意，大版本等 3 个月 |
| **iOS** | 非老旧机型推荐更新 |
| **Android** | 大版本前上论坛看看反馈 |
| **Linux** | 滚动发行版随更（如 Arch）；LTS 发行版选安全更新（如 Ubuntu） |

Linux 的更新方式跟其他系统不一样。滚动发行版（Arch Linux、Manjaro）采用"小步快跑"模式，每周甚至每天都有更新，不会有"大版本跳变"。LTS 发行版（Ubuntu LTS、Debian）类似 Windows——每两年一个大版本，平时只推送安全更新。

**更新后变卡的两个原因：**
1. 新系统主要针对新设备优化，老机型跑新系统会吃力
2. 更新后系统需要重建文件索引，持续几小时到一两天，期间会卡。等索引建完就恢复

**更新前准备：** 接电源 → 关掉运行中的程序 → 备份重要文件 → 选你不用电脑的时间段

## 硬盘与分区

### C 盘 D 盘不是物理存在的

你的电脑里可能只有一个物理硬盘，但被分成了多个**逻辑分区**。C 盘、D 盘是 Windows 给不同分区起的"门牌号"。

```
一个物理硬盘（500GB）
  ├── C 盘分区（200GB）——装系统 + 软件
  └── D 盘分区（300GB）——装文件
```

这不是两块硬盘。你从 C 盘复制文件到 D 盘，数据还在同一块盘上——只是从左边移到右边。真正的"跨硬盘复制"是复制到 U 盘或另一个物理硬盘上才有提速效果。

### 为什么 C 盘满了会卡

硬盘需要留出约 15-20% 的空白空间作为缓存和临时文件暂存区。C 盘满了，系统就没地方写临时文件——所以会卡。

**清理 C 盘的顺序：**

1. 运行磁盘清理（搜索"磁盘清理"）——清掉系统临时文件、缩略图缓存
2. 清空回收站和下载文件夹（下载文件夹是重灾区，建议每月清一次）
3. 把文档、桌面、下载文件夹移到 D 盘（右键文件夹 → 属性 → 位置 → 移动）
4. 关闭休眠（如果不用的的话，管理员运行 CMD：`powercfg -h off`，可以释放几 GB）
5. 清理旧系统更新文件（设置 → 系统 → 存储 → 临时文件 → "以前的 Windows 安装"）

### Linux 的分区方式

Linux 跟 Windows 完全不同。它没有 C 盘 D 盘的概念——根目录是 `/`，下面挂载不同的分区：

```
/
├── /boot     —— 内核和引导文件（通常单独一个分区）
├── /home     —— 用户文件（相当于 C 盘的用户文件夹，建议单独分区）
├── /var      —— 日志和缓存
└── /usr      —— 软件安装位置
```

好处：重装系统时只要保留 `/home` 分区的数据不动，其他分区随便格式化，用户文件不会丢。

## 显示设置

### 夜间模式

减少蓝光确实有助于睡眠，但屏幕蓝光的影响远小于房间灯光。夜间模式的主要作用是降低屏幕亮度和色温，让你在暗光下更舒服。

各平台开启：
- **iPhone：** 设置 → 显示与亮度 → 夜览，可设日落到日出自动开启
- **安卓：** 设置 → 显示 → 护眼模式（各品牌名称不同：华为叫"护眼模式"，小米叫"纸间模式"，三星叫"蓝光滤镜"）
- **Windows：** 设置 → 系统 → 显示 → 夜间模式
- **macOS：** 系统设置 → 显示器 → 夜览

### 深色模式

OLED 屏幕下纯黑背景省电 30-40%，深灰省电 10-15%。LCD 屏幕基本不省电。

暗光下深色模式确实更舒适，但长时间阅读深色文字因人而异。建议设自动切换——日间浅色、夜间深色。如果某个 APP 深色模式适配不好（微信早期版本就是反面教材），可以临时切回浅色。

### 分辨率与缩放

一个常见误解：分辨率不是越高越好。分辨率太高字太小，眼睛反而累。最佳组合是原生分辨率 + 合理缩放：

| 设备 | 推荐分辨率 | 推荐缩放 |
|:---|:---|:---|
| 13 寸笔记本 | 2560×1600 | 125-150% |
| 24 寸显示器 | 1920×1080 | 100% |
| 4K 电视当显示器 | 3840×2160 | 200-250%（但老软件可能模糊） |

## 隐私权限

**基本原则：不给不必要的权限。**

一个手电筒 APP 要求读取通讯录——没有合理理由，不给。如果功能因此用不了，APP 会主动提示你需要什么权限，那时再按需开启。

| 权限 | iOS 管理 | 安卓管理 | 建议 |
|:---|:---|:---|:---|
| **位置** | 精确/大致/永不 | 使用时/仅限此次/拒绝 | 地图/外卖给精确，其他给大致或拒绝 |
| **照片** | 全部/选中的/无 | 所有文件/选中的/拒绝 | 修图 APP 给全部，其他给选中的 |
| **相机** | 是/否 | 是/否 | 除了拍照/扫码 APP，其他一律否 |
| **麦克风** | 是/否 | 是/否 | 除了通话/录音 APP，其他一律否 |
| **通讯录** | 是/否 | 是/否 | 除了通讯类 APP，其他一律否 |

**iOS 必关：** 设置 → 隐私 → 跟踪 → 关闭"允许 APP 请求跟踪"；分析与改进 → 全关；Apple 广告 → 关个性化广告

**安卓必关：** 谷歌 → 广告 → 关广告个性化；位置信息 → 关 Wi-Fi 和蓝牙扫描；各 APP 的权限逐项审查

**Windows 必关：** 设置 → 隐私与安全性 → 常规 → 全关；诊断与反馈 → 选"基本诊断数据"

**定位精度三个等级：**
- 精确位置 → 地图、外卖需要
- 大致位置 → 天气、点评类 APP 够用
- 关闭 → 手电筒、计算器不需要位置

## 声音设置

常见困扰：连了蓝牙耳机声音从外放出来、插了耳机微信语音从听筒出来、外接显示器声音不知道从哪出来。

**Windows：** 右下角声音图标右键 → 声音设置 → 输出设备，选择正确的设备。不同程序用不同输出设备需要第三方工具 EarTrumpet。

**macOS：** 按住 Option 键点菜单栏音量图标，直接选择输出设备。

**iOS：** 控制中心 → 点击音频卡片 → 选择设备。AirPods 会在 iPhone 和 Mac 间自动切换——有时很方便，有时很烦。

**通知音个性化：** 安卓可以在应用管理里给不同通知类型设置不同声音。iOS 限制较多，只能用系统提供的通知音。

## 辅助功能

辅助功能不只是为特殊人群设计的。下面几个对日常使用也有用：

**放大镜：**
- iPhone：设置 → 辅助功能 → 缩放 → 开启（三指双击放大）
- 安卓：设置 → 辅助功能 → 放大功能 → 开启
- 用途：看小字、检查屏幕坏点、当临时放大镜

**颜色滤镜：**
- iPhone：设置 → 辅助功能 → 显示与文字大小 → 颜色滤镜
- 色盲模式帮助区分颜色，灰度模式让屏幕变黑白——戒手机效果很好

**语音控制：**
- 手上忙着别的事时用语音操作手机
- 苹果和安卓都有，各品牌名字不同

**长辈模式：**
- iPhone 引导式访问：锁定在当前 APP，防止误操作
- 安卓简易模式：图标变大、功能简化、字体放大
- 通用技巧：桌面只留最常用的 APP、关闭手势操作、开启勿扰模式自动规则

## 备份

数据丢失的原因排名：手滑删除 35%、设备损坏 25%、被盗 20%、系统崩溃 15%、勒索病毒 5%。

**3-2-1 备份法则：** 3 份数据、2 种介质、1 份异地。

各平台操作：

**iPhone：** iCloud 自动备份。5GB 不够用，建议 6 元/月升 50GB。
**安卓：** 品牌云服务 + 谷歌备份。照片定期拷到电脑。
**Windows：** 设置 → 更新与安全 → 备份。重要文件夹移到 D 盘。
**macOS：** 时间机器（Time Machine）——外置硬盘插上自动备份，macOS 最好的功能。

## 重装系统

### 什么时候该重装

- 系统崩溃无法正常启动
- 中了严重病毒杀不干净
- 用了三年以上积累太多垃圾
- 卖电脑前清空数据

**不该重装的时候：** 只是有点卡（先清理试试）、只是某个软件出问题（重装那一个软件就行）。

### 做启动 U 盘

装系统的第一步是做一个启动 U 盘：

**Windows：** 下载微软官方"媒体创建工具"（Media Creation Tool）→ 插入 8GB+ U 盘 → 运行工具选择"为另一台电脑创建安装介质"→ 选 U 盘 → 等它下完写进去

**更通用的方案：** 用 Ventoy（一个开源工具）→ 把 ISO 镜像直接拖进 U 盘 → 启动时选要装哪个系统。一个 U 盘能同时装 Windows、Ubuntu、PE 工具盘。

**Linux：** 下载 Ubuntu 或你选择发行版的 ISO → 用 Rufus 或 Ventoy 写入 U 盘

### 重装流程

1. 插上启动 U 盘 → 重启 → 进 BIOS 改启动顺序（U 盘排第一）
2. 保存退出 → 电脑从 U 盘启动 → 进入安装界面
3. 选"自定义安装"（不要选"升级"）
4. **分区选择这一步要小心：** 选择正确的分区，不要误格式化有数据的分区
5. 等待安装完成 → 拔掉 U 盘 → 重启
6. 进系统后装驱动（Windows 会自动更新大部分，显卡驱动去官网下）

## 蓝屏与崩溃简史

Windows 蓝屏的演变史，某种程度上也是操作系统成熟度的演变史。

**Windows 95/98：** 经典蓝屏，蓝底白字加一堆十六进制代码。用户反应：恐慌→拍照发论坛→等大神解答。

**Windows XP：** 加入了停止码和最著名的 `STOP 0x0000007B`（硬盘模式错误，装系统时常见）。很多人的第一次蓝屏体验。

**Windows 8/10/11：** 蓝屏变浅蓝色，加了二维码和 `:(` 表情。用户反应："哦，又蓝了"→ 重启。

::: info 蓝屏为什么是蓝色
据说是早期 Windows 开发团队的某个工程师显示器偏蓝，他觉得蓝色"看起来比较 calm"。这个配色就这样定型了。

Windows 95 时代的蓝屏还会发出一声"嘟"——那是程序员把调试提示音误留在了正式版里，结果这个 bug 延续到 Windows 10 才被移除。
:::

### macOS 和手机的崩溃画面

macOS 崩溃时显示的是"五国语言"错误——屏幕上用五种语言写着"你需要重启计算机"。比 Windows 蓝屏"优雅"一些，但本质一样：系统彻底挂了。苹果用户见得少，所以见到时格外震惊——"苹果也会崩溃？"

iPhone 卡住时是"转圈圈"（白苹果 + 进度条），通常发生在系统更新失败时。安卓的"无限重启"（开机动画 → 黑屏 → 循环）通常是因为系统文件损坏。

任何系统都会崩溃。遇到崩溃时最关键的不是"为什么崩溃了"——而是"我上次备份是什么时候"。

## 常见问题

**Q：更新卡在某个进度不动了怎么办？**

A：大版本更新可能需要 1-2 小时。超过 3 小时纹丝不动再强制重启。

---

**Q：恢复出厂设置后数据还能找回来吗？**

A：取决于你有没有备份。有备份就能恢复，没有就没了。恢复出厂设置前务必检查备份。

---

**Q：UEFI 和 BIOS 有什么区别？**

A：UEFI 是 BIOS 的现代替代品。BIOS 只能从 MBR 启动（最大支持 2TB 硬盘、最多 4 个主分区），UEFI 使用 GPT 分区表（支持更大硬盘、更多分区）。2015 年后的电脑基本都是 UEFI。进设置的方法一样——开机按 F2/Del。

---

**Q：Linux 适合日常使用吗？**

A：看你的需求。编程、写文档、上网、看视频——完全可以。打游戏（Steam 有 Proton 兼容层，但部分游戏不行）、用 Adobe 全家桶、用某些学校指定的软件——可能会有兼容性问题。建议先在虚拟机里体验 Ubuntu，觉得顺手再考虑双系统。
